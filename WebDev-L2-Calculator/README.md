# 🧮 Calculator Pro

A modern and feature-rich calculator built using **HTML5, CSS3, and Vanilla JavaScript**. This project goes beyond a basic calculator by providing a premium glassmorphism UI, scientific functions, history tracking, keyboard support, theme switching, and memory operations.

## ✨ Features

### 🧮 Basic Operations

* Addition (+)
* Subtraction (−)
* Multiplication (×)
* Division (÷)
* Decimal support
* Operator chaining
* Division-by-zero error handling

### 🔬 Advanced Functions

* Percentage (%)
* Positive/Negative (±)
* Square Root (√)
* Square (x²)
* Pi (π)

### 🧠 Memory Functions

* MC (Memory Clear)
* MR (Memory Recall)
* M+ (Memory Add)
* M− (Memory Subtract)

### 📜 History

* Stores previous calculations
* Clear history option
* History persists after page refresh using Local Storage

### 🌙 Theme Support

* Dark Mode
* Light Mode
* Theme preference saved automatically

### ⌨️ Keyboard Support

* Numbers (0–9)
* Decimal point (.)
* Operators (+, −, ×, ÷)
* Enter (=)
* Backspace (Delete)
* Escape (Clear)

### 📋 Extra Features

* Copy result to clipboard
* Live digital clock
* Responsive design
* Smooth animations
* Glassmorphism interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)
* Local Storage API

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/yourusername/calculator-pro.git
```

2. Open the project folder:

```bash
cd calculator-pro
```

3. Run the project by opening `index.html` in your browser.
